## 長い説明（1000字以内）

### Fibro — 標準エクスプローラーを超えるファイル管理ツール

Fibro は Windows の標準ファイルエクスプローラーを置き換える高速・高機能なファイル管理アプリケーションです。

#### 主な機能

**📋 タブ・デュアルペイン**
- 複数フォルダを同時管理（タブ / 左右2画面）
- Ctrl+T で新規タブ、F9 でデュアルペイン切替
- タブ状態は自動保存・復元

**🔍 強力な検索**
- ファイル名・本文内容・Excel セル値を同時検索
- FTS5 インデックスで瞬間検索（最大10倍高速）
- ワイルドカード対応（`*.md` など）

**✏️ 一括リネーム**
- 検索→置換、正規表現、連番付与（`${n}`）
- 大小文字変換、衝突自動回避
- ライブプレビューで安全実行、Ctrl+Z で Undo

**🖼️ サムネイル＆プレビュー**
- 詳細ビュー / サムネイル表示の切替（Ctrl+Shift+T）
- Space キーで画像・テキスト・ファイル情報をプレビュー

**⭐ その他の生産性機能**
- お気に入いフォルダ（階層構造・D&D 並び替え）
- 最近 / よく使うフォルダ自動表示
- マウス戻る・進むボタン対応
- ターミナル / PowerShell / VS Code から直接開く

#### 堅牢性

- ネットワークドライブのタイムアウト処理（UI フリーズなし）
- 260 文字超のロングパス対応（Windows）
- ファイル衝突時の多段階選択ダイアログ
- 設定ファイルの自動保護

#### プライバシー

- 個人情報収集なし
- ローカルファイルのみ処理
- 外部サーバーへの送信なし
- すべての設定データはマシン内に保存

#### 動作環境

- Windows 10 1809 以降
- Python 不要（exe 形式で配布）
- インストール・管理者権限不要
